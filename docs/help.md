# 帮助

## 主页

本网站是一个在线诗歌PPT浏览工具。主页是网站的入口，点击导航栏中的站点名称即可进入对应站点的目录页。

## 内容浏览页面

进入目录页后，点击任意文件夹即可进入内容浏览页面，逐张查看文件夹内的图片。

### 如何翻页

- **鼠标点击**：点击图片页面任意位置（非导航控件区域）即可翻到下一页；在最后一页再次点击可返回目录。
- **鼠标滚轮**：向上滚动切换到上一张，向下滚动切换到下一张。
- **键盘操作**：按 **右箭头键**（→）切换到下一张，按 **左箭头键**（←）切换到上一张。
- **触屏滑动**：在手机或平板上，向左滑动切换到下一张，向右滑动切换到上一张。

### 如何使用预览栏

将鼠标移动到屏幕左侧边缘，会自动滑出预览栏。预览栏中显示了当前文件夹内所有图片的缩略图，点击任意缩略图可快速跳转到对应页面。鼠标移出预览栏后会自动隐藏。

### 如何使用全屏模式

将鼠标移动到屏幕顶部中央区域，会出现全屏切换按钮。点击按钮可进入或退出全屏模式，获得更好的浏览体验。

### 页码输入

点击底部的页码指示器（如"5 / 20"），可以输入目标页码直接跳转到指定页面，按回车键确认。
