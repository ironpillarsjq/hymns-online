# 诗歌在线浏览

在线诗歌图片浏览工具，无需安装，打开网页即可使用。

## 使用方法

### 目录页

打开网页后进入目录页，这里列出了所有可浏览的诗歌文件夹。

**切换布局**：页面右侧有一个切换按钮，可以在两种布局间切换。

- **列表布局**：每个诗歌文件夹占一行，纵向排列，适合文件夹数量较少时使用
- **网格布局**：文件夹以网格形式展示，每 10 个为一小组，每 10 小组为一大组。每行显示 5 个按钮，适合大量文件夹时快速浏览

**滚动页面**：目录页的标题和内容一起上下滚动，方便随时查看当前页面标题。

**点击任意诗歌文件夹**即可进入阅读器浏览其中的图片。

### 图片阅读器

进入阅读器后，图片铺满整个屏幕，始终保持原有的宽高比例。

**翻页方式**：
- 点击页面右侧区域 → 下一张
- 点击页面左侧区域 → 上一张
- 键盘方向键：← 上一张 / → 下一张
- 鼠标滚轮：向下翻下一页，向上翻上一页
- 触控设备：左右滑动切换

**底部导航栏**：屏幕底部有一个半透明栏，包含"目录"返回按钮和上一页/下一页按钮。鼠标不动 3 秒后自动隐藏，移动鼠标到屏幕底部重新显示。

**页码指示器**：屏幕底部中央显示当前页码（如 "5 / 20"）。点击页码可切换为输入框，直接输入数字后按回车可跳转到指定页。

**左侧预览栏**：移动鼠标到屏幕最左边，会弹出缩略图列表，显示所有图片的预览。点击任意缩略图可直接跳转到该图片，当前页的缩略图有白色边框标记。

**首末页提示**：已经在第一页或最后一页时继续翻页，屏幕中央会显示"再次点击退出"或"再次点击返回目录"的提示，按提示再次操作即可返回目录页。

**全屏模式**：移动鼠标到屏幕顶部中央，会显示一个全屏按钮。点击后浏览器进入全屏模式，再次操作可退出全屏。

## 如何添加自己的诗歌

将图片放入 `public/data/` 目录下，按照以下结构组织：

```
public/data/
└── <站点标题>/          # 站点的名称（显示在页面顶部）
    ├── <文件夹1>/        # 一个诗歌文件夹
    │   ├── 001.jpg
    │   ├── 002.jpg
    │   └── ...
    ├── <文件夹2>/
    └── ...
```

- **站点标题**：`public/data/` 下的第一个文件夹名称就是网站标题
- **诗歌文件夹**：每个文件夹是一组图片，文件夹名称会显示在目录页中
- **图片文件名按数字排序**，如 `001.jpg`、`002.jpg` …，格式支持 jpg、png、webp、gif、bmp

## 本地启动（开发者）

```bash
npm install
npm run dev          # 启动开发服务器，打开 http://localhost:5173
npm run build        # 构建到 docs/ 目录
npm run preview      # 预览构建产物
```
