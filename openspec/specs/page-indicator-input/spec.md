## ADDED Requirements

### Requirement: 点击页码切换为输入模式

系统 SHALL 在用户点击底部页码指示器时，将页码显示切换为数字输入框，预填当前页码并自动聚焦全选。

#### Scenario: 点击页码指示器进入输入模式
- **WHEN** 用户在页码指示器上点击
- **THEN** 显示文本 "3 / 12" 切换为输入框，输入框中预填 "3" 且文本被全选
- **AND** 输入框自动获取焦点
- **AND** 组件向父组件发出 `editstart` 事件

### Requirement: 输入页码跳转

系统 SHALL 支持用户输入 1 到总页数范围内的整数页码，确认后跳转到对应页面。

#### Scenario: 输入有效页码按 Enter 跳转
- **WHEN** 用户输入 "8" 且总页数为 12，按 Enter 键
- **THEN** 组件退出输入模式，恢复文本显示 "8 / 12"
- **AND** 向父组件发出 `jump-to` 事件，参数为 8

#### Scenario: 输入有效页码后失焦跳转
- **WHEN** 用户输入 "5" 且总页数为 12，点击输入框外部区域（失焦）
- **THEN** 行为与按 Enter 一致——跳转到第 5 页并退出输入模式

#### Scenario: 输入超出范围的页码
- **WHEN** 用户输入 "20" 且总页数为 12，按 Enter 或失焦
- **THEN** 组件退出输入模式，恢复原有页码显示（不跳转，不发出 `jump-to` 事件）

#### Scenario: 输入非数字内容
- **WHEN** 用户输入 "abc" 且总页数为 12，按 Enter 或失焦
- **THEN** 组件退出输入模式，恢复原有页码显示（不跳转，不发出 `jump-to` 事件）

#### Scenario: 输入 0 或负数
- **WHEN** 用户输入 "0"（或 "-3"）且总页数为 12，按 Enter 或失焦
- **THEN** 组件退出输入模式，恢复原有页码显示（不跳转，不发出 `jump-to` 事件）

### Requirement: Esc 取消输入

系统 SHALL 支持按 Esc 键取消输入操作，恢复原有页码显示且不执行跳转。

#### Scenario: 按 Esc 取消输入
- **WHEN** 用户处于输入模式，按 Esc 键
- **THEN** 组件退出输入模式，恢复原有页码显示
- **AND** 向父组件发出 `editcancel` 事件
- **AND** 不发出 `jump-to` 事件

### Requirement: 输入期间暂停自动隐藏

系统 SHALL 在输入模式期间通知父组件暂停自动隐藏计时器，确认或取消后恢复。

#### Scenario: 进入输入模式时暂停计时
- **WHEN** 用户点击页码指示器进入输入模式
- **THEN** 父组件收到 `editstart` 事件后暂停 3 秒自动隐藏计时器

#### Scenario: 确认跳转后恢复计时
- **WHEN** 用户确认输入完成跳转
- **THEN** 父组件重新启动 3 秒自动隐藏计时器

#### Scenario: 取消输入后恢复计时
- **WHEN** 用户按 Esc 取消输入
- **THEN** 父组件重新启动 3 秒自动隐藏计时器
