## 新增需求

### 需求：全屏按钮显示与隐藏

系统应在屏幕顶部中央提供一个圆形全屏切换按钮。按钮在鼠标光标靠近屏幕顶部时显示，光标离开该区域后隐藏。按钮的显示与隐藏应带有平滑的淡入淡出过渡效果。

#### 场景：鼠标靠近时显示按钮
- **当** 用户鼠标光标进入屏幕顶部 60px 范围内
- **则** 全屏按钮应以淡入动画显示，opacity 从 0 过渡到 1

#### 场景：鼠标离开后隐藏按钮
- **当** 用户鼠标光标离开屏幕顶部 60px 范围
- **则** 全屏按钮应以淡出动画隐藏，opacity 从 1 过渡到 0

#### 场景：浏览器不支持全屏时隐藏按钮
- **当** 浏览器不支持 Fullscreen API（`document.fullscreenEnabled` 为 false）
- **则** 全屏按钮应始终隐藏（`display: none`）

### 需求：全屏切换功能

系统应在用户点击全屏按钮时，调用浏览器 Fullscreen API 进入或退出全屏模式。按钮图标应根据当前全屏状态显示进入全屏或退出全屏的图标。

#### 场景：点击按钮进入全屏
- **当** 用户点击全屏按钮且当前未处于全屏模式
- **则** 系统应调用 `document.documentElement.requestFullscreen()` 进入全屏模式

#### 场景：点击按钮退出全屏
- **当** 用户点击全屏按钮且当前处于全屏模式
- **则** 系统应调用 `document.exitFullscreen()` 退出全屏模式

### 需求：进入全屏时短暂显示按钮

系统应在用户刚进入全屏模式时短暂显示全屏按钮，作为状态确认，随后自动隐藏。

#### 场景：进入全屏后短暂显示
- **当** 浏览器进入全屏模式（通过按钮或其他方式）
- **则** 全屏按钮应显示 2 秒后自动淡出隐藏

#### 场景：短暂显示期间被鼠标交互打断
- **当** 全屏按钮正在短暂显示中，且用户鼠标移入按钮区域
- **则** 自动隐藏的定时器应被取消，按钮保持显示直到用户鼠标离开

### 需求：按钮视觉样式

全屏按钮应为圆形（直径 48px），位于屏幕顶部中央，背景色为主色，图标为白色。

#### 场景：按钮基础样式
- **当** 按钮处于可见状态
- **则** 按钮应为直径 48px 的圆形，水平居中于视口，垂直位置距顶部 16px，背景色为 `--color-primary`（#626C83），图标颜色为 `--color-background`（#FFFFFF）

#### 场景：按钮 z-index
- **当** 按钮渲染时
- **则** 按钮的 z-index 应足够高（如 9999），确保始终在所有页面内容之上
