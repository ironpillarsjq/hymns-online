## 新增需求

### 需求：CSS 变量颜色系统

系统应通过 CSS 自定义属性定义颜色变量，所有组件样式引用这些变量而非硬编码颜色值。

#### 场景：全局颜色变量定义
- **当** 应用初始化时
- **则** CSS 变量应包含：`--color-primary: #626C83`、`--color-secondary: #888FA0`、`--color-light: #D4D6DC`、`--color-very-light: #FAFAFA`、`--color-background: #FFFFFF`、`--color-text: #797979`、`--color-text-light: #AEB3BE`、`--color-error: #e57373`

### 需求：页面背景色

系统所有页面背景应使用 `#FFFFFF`（纯白），替代原有的深色背景。

#### 场景：TOC 页面背景
- **当** 用户访问目录页
- **则** 页面背景应为白色 `#FFFFFF`

#### 场景：PDF 阅读器页面背景
- **当** 用户访问 PDF 阅读器页面
- **则** 阅读器外层容器背景应为白色 `#FFFFFF`，内部 PDF 渲染区域保持深色 `#0d0d1a`

### 需求：文字颜色

系统所有文字应使用 `#797979`（深灰色），替代原有的浅色文字。

#### 场景：正文文字颜色
- **当** 页面渲染文本内容时
- **则** 文字颜色应为 `#797979`

#### 场景：次要文字颜色
- **当** 页面渲染辅助信息（如加载提示、空状态说明）
- **则** 文字颜色应为 `#AEB3BE`

### 需求：主题强调色

系统的交互元素（按钮、链接、hover 效果）应使用主色 `#626C83`。

#### 场景：按钮主题色
- **当** 导航控件按钮渲染时
- **则** 按钮背景应为 `#626C83`，文字为白色 `#FFFFFF`

#### 场景：列表项 hover
- **当** 用户鼠标悬停在目录列表项上
- **则** 列表项边框颜色变为 `#626C83`

### 需求：卡片与分割线

系统的卡片组件背景应使用浅灰色，分割线使用更浅的灰色。

#### 场景：TOC 列表项背景
- **当** 目录页渲染 PDF 列表项时
- **则** 列表项背景为 `#FAFAFA`，边框为 `#D4D6DC`

#### 场景：分割线颜色
- **当** 页面中存在分割线元素时
- **则** 分割线颜色为 `#D4D6DC`
