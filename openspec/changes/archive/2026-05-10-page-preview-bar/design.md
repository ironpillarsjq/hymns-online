## Context

用户需要在 PDF 阅读器中快速浏览和跳转到任意页面。当前阅读器仅支持逐页导航，缺乏全局页面概览和快速跳转能力。

此项目为纯前端 Vue 3 单页应用，使用 PDF.js 渲染 PDF。已有的浮层模式（PageIndicator、NavControls）通过 `position: fixed` + 3 秒隐藏定时器实现。现有代码在 `PdfViewerPage.vue` 中集中管理阅读器状态（当前页、总页数、控件显隐）。

## Goals / Non-Goals

**Goals:**
- 在 PDF 阅读器页面左侧新增一个半透明的页面缩略图预览栏
- 鼠标靠近屏幕左边缘（约 30px 内）时触发滑入显示
- 预览栏纵向列出所有 PDF 页面的缩略图，支持滚轮滚动
- 点击任意缩略图立即跳转到对应页面，预览栏随之隐藏
- 预览栏内的当前页缩略图有高亮标识
- 与现有的键盘导航、触控滑动等控件行为兼容

**Non-Goals:**
- 不改变 PDF 主体渲染逻辑（PdfRenderer 保持不变）
- 不在移动端触发（触屏设备无 hover 概念，避免误触）
- 不支持缩略图拖拽排序或缩放
- 不修改 manifest 或数据管道

## Decisions

### 1. 新增独立组件 `PagePreviewBar.vue`

**决策**: 新建 `src/components/PagePreviewBar.vue`，由 `PdfViewerPage.vue` 引入。

**理由**: 遵循现有项目的组件拆分模式（PageIndicator、NavControls 均为独立组件），保持 PdfViewerPage 的协调层职责清晰。

**替代方案**: 在 PdfViewerPage 中内联实现 — 不采用，会使该文件过于庞大。

### 2. 触发区域：30px 左边缘热区

**决策**: 通过 `mousemove` 事件监听鼠标 X 坐标，当 `clientX <= 30` 时显示预览栏。使用 CSS `transition` 实现滑入滑出动画。

**理由**: 
- 30px 是常见的边缘触发宽度，不会干扰主体区域操作
- `mousemove` 是唯一可行的鼠标位置感知方式（纯 CSS 无法实现"靠近边缘触发"效果）
- 与现有的 `mousemove` 监听（用于重置隐藏定时器）可合并

**替代方案**: 
- 纯 CSS `:hover` 在固定热区上 — 不采用，因为热区不可见时用户不知其存在，且 `:hover` 无法实现"靠近边缘"的渐进触发。
- 始终显示一个小的折叠手柄 — 可考虑作为后续迭代，当前保持极简。

### 3. 缩略图渲染策略：全部预渲染

**决策**: PDF 加载完成后，一次性渲染所有页面的缩略图到离屏 canvas，再绘制到预览栏中。

**理由**:
- 歌词/诗歌类 PDF 页码通常较少（<20 页），全量渲染性能开销可接受
- 避免滚动时异步加载导致的闪烁

**替代方案**: 按需渲染（仅渲染滚动可视区内的缩略图） — 如未来支持大型 PDF 可升级，当前过度设计。

### 4. 缩略图尺寸：固定宽度 120px

**决策**: 缩略图固定宽度 120px，高度按页面比例自适应。

**理由**: 120px 宽度足以辨认页面内容，同时不会占据过多屏幕空间。渲染时使用低分辨率 scale 以节省内存。

### 5. 预览栏整体宽度 150px

**决策**: 预览栏固定宽度 150px，半透明背景（`rgba(0, 0, 0, 0.85)`），z-index 略高于导航控件（如 z-index: 15）。

### 6. 显示/隐藏行为

**决策**: 
- **显示**: 鼠标进入左边缘 30px 热区 → 立即显示（`transition: transform 0.2s` 从左侧滑入）
- **隐藏**: 鼠标移出预览栏区域 → 延迟 1 秒后隐藏（比底部控件的 3 秒更快，因为预览栏更碍眼）
- **跳转后隐藏**: 点击缩略图跳转 → 立即隐藏

### 7. 与现有 auto-hide 机制的集成

**决策**: 预览栏的显示/隐藏独立于现有的 `controlsVisible` 定时器。但导航到新页面时（包括通过预览栏跳转），继续重置底部控件的隐藏定时器。

**理由**: 预览栏和底部控件有不同的显示逻辑（一个基于鼠标位置，一个基于活动），强行耦合会使逻辑复杂。

## Risks / Trade-offs

- **[性能] 全量缩略图渲染对于 50+ 页的 PDF 可能导致短暂延迟** → 当前场景页码少，风险低。如未来页码增多，可升级为 IntersectionObserver 按需渲染。
- **[UX] 30px 热区在极端窄窗口下可能太宽** → 当窗口宽度 < 800px（移动端）时，禁用此功能，仅桌面端使用。
- **[冲突] 鼠标在左边缘附近操作时，预览栏可能意外触发** → 1 秒隐藏延迟可减少闪烁感；且预览栏不阻挡点击穿透（使用 `pointer-events: none` 在热区区域，仅预览栏本身可交互）。
